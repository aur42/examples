# Overview

This repository contains an example script demonstrating a basic recurring flat rate pricing model. 

# Installation

Run once:

```bash
yarn
```

# Setup

1. Set your Sphere ApiKey in your environment variables.

```bash
export API_KEY="<SPHERE_API_KEY>"
```

# Run

To run the script:

```bash
yarn run dev
```
